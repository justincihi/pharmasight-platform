REST API Examples

Access all RDKit functionality via HTTP endpoints.

1. Health Check

curl http://localhost:5000/api/rdkit/health # Response: { "status": "ok", "rdkit_version": "2025.09.1", "available_endpoints": [...] }

2. Calculate Properties

curl -X POST http://localhost:5000/api/rdkit/properties \ -H "Content-Type: application/json" \ -d '{"smiles": "CC(=O)O"}' # Response: { "molecular_weight": 60.05, "logP": 0.09, "num_h_donors": 1, "num_h_acceptors": 2, "lipinski_violations": 0 }

3. Generate Visualization

curl -X POST http://localhost:5000/api/rdkit/visualize \ -H "Content-Type: application/json" \ -d '{"smiles": "c1ccccc1", "format": "png", "size": [400, 400]}' # Response: { "format": "png", "data": "data:image/png;base64,iVBORw0KG..." }

4. Calculate Similarity

curl -X POST http://localhost:5000/api/rdkit/similarity \ -H "Content-Type: application/json" \ -d '{"smiles1": "CC(=O)O", "smiles2": "CCC(=O)O"}' # Response: { "similarity": 0.417, "method": "morgan" }

5. Generate Analogs

curl -X POST http://localhost:5000/api/rdkit/analogs \ -H "Content-Type: application/json" \ -d '{"smiles": "CC(=O)O", "num_analogs": 5}' # Response: { "parent_smiles": "CC(=O)O", "num_analogs": 5, "analogs": [ { "smiles": "CCC(=O)O", "similarity": 0.85, "molecular_weight": 74.08, "logP": 0.33 } ] }

Start the API Server

cd /home/user/pharmasight-platform/src /root/miniconda/envs/pharmasight/bin/python rdkit_api.py

Server will start on http://localhost:5000
